# Instruções para o Claude Code

## Segredos — ficheiros proibidos de ler

Nunca ler, citar, nem sugerir alterações ao conteúdo dos seguintes ficheiros:

- `.env.local`
- `.env`
- `.env.*`
- `.dev.vars`

Se precisares de saber que variáveis existem, consulta `.env.local.example` ou `wrangler.toml` — nunca o ficheiro real com valores.
